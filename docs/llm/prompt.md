# 知识学习类

```bash
我目前是[身份/学习阶段]，正在学习[主题/领域]，已经了解了[已有知识基础]。
我希望深入理解[具体概念/原理/技能]。
请以[深度/广度]讲解，使用[简单/专业]的语言，最好能提供[实例/类比]来辅助理解。
我更倾向于通过[学习方式：系统化/问答式/案例式]来学习，
如果有[特定困惑]请重点解释
请以[结构化方式/图文描述/步骤说明]呈现，便于我系统掌握。
```

# 信息查询类

```bash
我需要了解关于[具体主题/问题]的信息。
这些信息将用于[用途：决策/研究/个人了解]。
请提供[最新/权威/全面]的信息，包括[关键数据/定义/观点/比较]。
我需要[基础概述/中等深度/专业深入]的信息
请以[列表/表格/要点总结/问答形式]呈现，并注明信息的[时效性/可靠性]。
```

# 优化翻译类

```bash
这是一份[文档类型：邮件/论文/广告/技术文档]，面向[目标受众]，主题是[主题内容]。
请帮我[翻译/润色/校对/改写]这段内容。
源语言是[源语言]，目标语言是[目标语言]，语言风格需要[正式/友好/专业/简洁]。
内容涉及[专业领域]，请确保相关术语准确。
请特别注意[文化适应性/专业术语/语法问题/表达自然度]并且[保持原意/调整表达/本地化改写]
```

# 教学辅助类

```bash
我是[教师/学生/家长]，，需要为[学生年龄/年级/水平]的[学科/主题】提供教学支持。我需要[设计课程/创建练习题/制作教案/评估学习/提供辅导]。
希望通过这个教学活动达到[具体学习目标/能力培养]。可用的教学时间是[时长]，教学环境是[课堂/线上/家庭]，学生基础是[基础情况]
希望采用[教学方法/互动方式]，强调[重点关注的能力/知识点]。请提供[详细教案/练习题及答案/评估标准/教学建议]格式为[结构化/图文并茂/步骤式]。
```

# 创意发散类

```bash
我正在为[项目/产品/活动/问题]寻求创意灵感。希望创作出能够[传达信息/解决问题/吸引受众/表达情感]的内容。
主要面向[年龄段/职业/兴趣群体]的受众。希望风格偏向[幽默/严肃/前卫/传统/温暖/冷静]。
需要考虑[预算/时间/技术/文化]等限制因素
请提供[数量]个多样化的创意方向，每个创意包含[核心理念/实施建议/预期效果]。
```

# 文件阅读综述类

```bash
我有[文档类型：报告/论文/合同/会议记录]，主题是[主题内容]，长度约[页数/字数]。请帮我[总结要点/提取关键信息/比较异同/分析观点]。特别关注文档中的[特定部分/数据/论点/结论/建议]。
请将结果整理为[摘要长度/结构形式]，保持[专业度/简洁度]。
这份综述将用于[决策参考/进一步研究/信息共享/学习理解]。处理时请注意[保留专业术语/简化复杂概念/突出时间线/强调因果关系]。
```
